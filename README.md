# 来云顶班 — 全国城市合伙人招商平台

同城日结兼职交付基础设施 · 城市合伙人招募工具

## 页面结构

| 页面 | 文件 | 内容 |
|------|------|------|
| 首页 | `index.html` | 项目概述、核心优势、运作流程、MVP验证 |
| 市场机会 | `market.html` | 市场全景、城市分布、渗透路径、四方共赢、三年财务预测 |
| 合作模式 | `model.html` | 三种合作模式、收费标准、收益模型、投资亮点、发展路线图 |
| 数据看板 | `kpi.html` | KPI考核机制、335城市KPI详情(Swiper+雷达图)、城市排名 |
| 全国布局 | `layout.html` | 中国地图、三年扩展策略、335城市完整列表、分布统计 |
| 常见问题 | `faq.html` | FAQ、盈利模式、商家入驻流程、合伙人条件、收益预期 |

## 技术栈

- 纯静态 HTML/CSS/JS，无构建工具
- [ECharts 5](https://echarts.apache.org/) — 图表可视化
- [Swiper 11](https://swiperjs.com/) — 城市KPI卡片滑动
- [DataV GeoJSON](https://datav.aliyun.com/) — 中国地图数据
- Apple 官网设计风格
